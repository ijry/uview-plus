# 登录界面

提示

页面模板源码免费开源，在uni-app的插件市场uview-plus的 示例项目 中，在右上角选择"使用 HBuilderX 导入示例项目" 或者 "下载示例项目ZIP"， 在HX运行项目即可看到和使用模板。

这个界面功能，为模仿美团APP的登录界面，包括输入手机号，获取验证码填写(uview-plus的MessageInput组件)等共两个界面。
如果开发者用到此模板，请自行修改相关的结构和样式等，以达到自己所需的效果。
