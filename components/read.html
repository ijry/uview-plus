uview-plus

多平台快速开发的UI框架

# 说明

uview-plus，是uni-app (opens new window)生态优秀的UI框架，全面的组件和便捷的工具会让您信手拈来，如鱼得水

# 特性

  • 兼容安卓，iOS，微信小程序，H5，QQ小程序，百度小程序，支付宝小程序，头条小程序
  • 60+精选组件，功能丰富，多端兼容，让您快速集成，开箱即用
  • 众多贴心的JS利器，让您飞镖在手，召之即来，百步穿杨
  • 众多的常用页面和布局，让您专注逻辑，事半功倍
  • 详尽的文档支持，现代化的演示效果
  • 按需引入，精简打包体积

# 安装

# npm方式安装
npm i uview-plus

# 快速上手

  1. main.js引入uview-plus库
// main.js
import uviewPlus from 'uview-plus';
app.use(uviewPlus);
  1. App.vue引入基础样式(注意style标签需声明scss属性支持)
/* App.vue */
<style lang="scss">
@import "uview-plus/index.scss";
</style>
  1. uni.scss引入全局scss变量文件
/* uni.scss */
@import "uview-plus/theme.scss";
  1. pages.json配置easycom规则(按需引入)
// pages.json
{
	"easycom": {
		"^u-(.*)": "uview-plus/components/u-$1/u-$1.vue"
	},
	// 此为本身已有的内容
	"pages": [
		// ......
	]
}

请通过快速上手 (opens new window)了解更详细的内容

# 使用方法

配置easycom规则后，自动按需引入，无需import组件，直接引用即可。

<template>
	<u-buton>按钮</u-buton>
</template>

请通过快速上手 (opens new window)了解更详细的内容
